Shop For Home E commerce site Full Stack Development (.NET + Angular) 
ShopForHome is an online e-commerce platform developed to allow users to browse, purchase, and manage home decor products through a web application.
The system supports two types of users: Customers and Administrators.
Customers can browse products, add them to cart, place orders, and maintain wishlists. 
Administrators can manage products, users, stock, and generate reports. 
The ShopForHome system provides a complete e-commerce platform for customers and administrators. 
The system is built using ASP.NET Core Web API for backend services and Angular for the frontend user interface, with SQL Server used for database management.
